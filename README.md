# Getting Started with the following steps

cp -rf react-ts-boilerplate-prj travel-list
cd travel-list
rm -rf .git
rm -rf node_modules
npm install
npm start
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
git init
git branch -m main
git add .
git commit -m "travel-list initial commit"
git remote add origin [git@bitbucket.org:Siecle/travel-list.git]
git push -u origin main
